# Pakistan Heatwave Atlas

A research project focused on the analysis of heatwave characteristics, trends, and exposure across Pakistan using climate reanalysis datasets.

## Objectives

* Identify heatwave events across Pakistan
* Analyze spatial and temporal variability
* Map heatwave hotspots
* Assess population exposure to extreme heat

## Data Sources

* ERA5 Reanalysis
* WorldPop Population Data

## Tools

* Python
* QGIS
* Pandas
* Xarray
* GeoPandas

## Project Status

Repository under development.
